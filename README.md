<h4>
Este es un material de uso pedagógico y educativo sobre aspectos introductorios al programa R
</h4>

Es un material de uso exclusivo y con fines pedagógicos y educativos para difusión y enseñanza del programa R, en especial para aquellos que no tienen fundamentos de programación.

Es un proyecto que pretende ir actualizándose e ir incorporando temas y ejemplos de forma que cualquiera pueda seguirlos y sirva como material de consulta rápida.

Hasta el momento, los temas trabajados son:

- Secuencias, Vectores y Matrices
- Data frames
- Paquetes o Librerías
- Importar una base de datos o fichero de datos
- Importar datos desde una web
- Estructuras de control:
   - if, for, while, switch
- Funciones

Este material se puede visualizar en este link del repositorio: https://estebanlab2021.github.io/intro_r/
