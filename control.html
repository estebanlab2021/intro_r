Las estructuras de control permiten controlar la manera o forma en que se ejecuta un determinado cdigo.

Las estructuras de control establecen condiciones sobre el cdigo. Por ejemplo, qu condiciones deben cumplirse para realizar una operacin o qu debe ocurrir para ejecutar una funcin.

Esto es de gran utilidad para determinar la lgica y el orden en que ocurren las operaciones, en especial al definir funciones.

Por lo general son estructuras de cdigo condicionales o iterativas (ciclos, bucles)

Las estructuras de control ms usadas en R son las siguientes:

Estructura de Control Descripcin
if, else Si, de otro modo
for Para cada uno en
while Mientras
break Interrupcin
next Siguiente
switch Opera segn valor

1.- if, else

El if(), es usado cuando deseamos que una operacin se ejecute nicamente cuando una condicin se cumple.

Por ejemplo:

if(4 > 3){
  print("4 es mayor que 3")
}
[1] "4 es mayor que 3"

Por su parte el else se usa para complementar al if. Indica qu ocurrir cuando la condicin no se cumple

por ejemplo:

if(4 > 5){
  print("4 es mayor que 5")
}else{
  print("4 es menor que 5")
}
[1] "4 es menor que 5"

Sin embargo, no se puede usar el if para recorrer un vector, porque slo se usar el primer elemento del vector para evaluar si la condicin es verdadera y lo dems ser ignorado.

Por ejemplo si usamos este condicional para que indique cuales valores de un vector son menores de 4:

if(1:10 < 4){
  print("Verdadero")
}

Nos arroja el siguiente mensaje:

Error en if (1:10 < 3) {: the condition has length > 1

Esto quiere decir, que no podemos ejecutar el if() para todos y cada uno de los valores de un vector. Para corregir esto, usamos el ifelse(), que se usa para recorrer un vector segn una condicin.

ifelse(1:10 < 4, TRUE, FALSE)
 [1]  TRUE  TRUE  TRUE FALSE FALSE FALSE FALSE FALSE FALSE FALSE

Aqu vemos cmo aparece el valor de TRUE en cada uno de los valores hasta el 3, de ah en adelante, aparece un FALSE, porque no se cumple la condicin

2.- FOR

La estructura for permite ejecutar un bucle (loop), realizando una operacin para cada elemento de un conjunto de datos.

La estructura base es:

for(elemento in objeto){
  operacin con el elemento
}

Esto quiere decir que: PARA cada elemento EN el objeto, haz la siguiente operacin.

El elemento dentro del parntesis se puede llamar de cualquier forma, por ejemplo, le podemos llamar i o hacer alusin a los elementos dentro de un vector, por ejemplo fruta si el vector es de frutas, pero el nombre del vector u objeto, debe ser exactamente el mismo nombre.

Ejemplo simple recorriendo un vector

for(i in 1:5){
   print(i)    
}
[1] 1
[1] 2
[1] 3
[1] 4
[1] 5

Vemos que este cdigo lo que hace es que recorre cada elemento del vector y lo imprime.

Ahora vamos a crear otro vector y luego usando el for(), vamnos a recorrerlo y a cada elemento le sumamos 1.

objeto <- 1:5  # vector con los nmeros de 1 al 5

for(elemento in objeto){
  print(elemento+1)  #imprimir en consola a cada elemento del vector sumado a 1
}
[1] 2
[1] 3
[1] 4
[1] 5
[1] 6

Observe que hemos cambiado el ndice i por elemento, y el cdigo se ejecuta de igual manera

Esto hace exactamente lo mismo si nombramos al ndice a

for(a in objeto){
  print(a+1)
}
[1] 2
[1] 3
[1] 4
[1] 5
[1] 6

Lo importante es mantener la coherencia dentro del cdigo.

Si nos interesa guardar el resultado de un for, lo podemos vectorizar en otro objeto:

vector <- NULL   #Creamos un vector vaco

Luego ejecutamos el for donde decimos que cada resultado sea guardo en el vector usando las reglas de la indexacin:

for(i in objeto){
  vector[i]<- i+1 #cada valor del objeto se le suma 1 y se guarda en el vector iniciando en el indice i, que en este caso ser el 1
}

Verificamos el resultado del vector

vector
[1] 2 3 4 5 6

3.- while

Este es un tipo de bucle que ocurre mientras una condicin es verdadera (TRUE). La operacin se realiza hasta que se llega a cumplir un criterio previamente establecido.

La estructura base del while es:

while(condicion) {
  operaciones
}

Esto quiere decir que: MIENTRAS esta condicin sea VERDADERA, realiza estas operaciones.

Ejemplo:

#Definimos dos objetos. Uno con un valor de 5 y otro con valor 0

umbral <- 5  
valor <- 0

#Aplicamos el while
while(valor < umbral) {    #establecemos la condicin
  print("Todava no.")     #se imprime este mensaje si se cumple la condicin
  valor <- valor + 1       #En cada ciclo sumamos 1 al valor
}
[1] "Todava no."
[1] "Todava no."
[1] "Todava no."
[1] "Todava no."
[1] "Todava no."

Este cdigo imprimir en la consola el texto Todava no. mientras el objeto valor sea menor que el umbral (esto porque en cada ciclo valor se aumenta en 1). Una vez que el objeto valor sea mayor que umbral, se detiene el proceso. Se observa que en cada ciclo se suma +1 al valor actual de valor y esa suma se guarda en el mismo objeto: valor <- valor + 1, es decir, en el 1er ciclo valor es igual a 0 , pero le estamos sumando 1, entonces, cuando empieza el 2do ciclo, valor es igual a 1 y as sigue hasta que valor llegue a 5.

Si no, se agrega esta instruccin: valor <- valor + 1, se crear un bucle infinito. Solo se puede detener presionando la tecla ESC, de otro modo, correr por siempre y puede llegar a congelar el equipo.

Veamos otro ejemplo:

x <- 1
while(x <=5){
  print(x)
  x <- x+1
}
[1] 1
[1] 2
[1] 3
[1] 4
[1] 5

Aqu se va a imprimir el valor de x hasta que este sea menor o igual a 5.

4.- break y next

break y next son palabras reservadas en R, no se puede asignarles nuevos valores y realizan una operacin especfica cuando aparecen o se usan en el cdigo. Ambas funcionan para for y while, pero dentro de un if() anidado.

  • break permite interrumpir un bucle.

  • next permite avanzar a la siguiente iteracin del bucle, saltndose la actual.

Usando break

Para interrumpir un bucle con break, necesitamos que se cumpla una condicin. Cuando esto ocurre, el bucle se detiene, aunque existan elementos a los cuales an podra aplicarse.

Por ejemplo:

Interrumpimos un for cuando i es igual a 3, aunque an queden 7 elementos en el objeto.

for(i in 1:10){
  if(i==3){
    print("Aqu se detiene")
    break   #implementamos la interrupcin
  }
  print(i)  #vamos imprimiendo los valores del vector
}
[1] 1
[1] 2
[1] "Aqu se detiene"

Observamos que se imprime el 1 y 2 , pero donde deba estar el 3, se imprimi el mensaje Aqu se detiene y se detuvo el ciclo.

Para interrumpir un while antes de se cumpla la condicin lo vemos en el siguiente ejemplo: implementamos la condicin que un numero sea mayor a 5, y cuando este tenga el valor de 15, se detiene.

numero <- 20  #definimos un objeto con el valor de 20

while (numero >5) {
  if(numero==15){
    print(paste("Se detiene cuando es igual a", numero))
    break #implementamos la interrupcin
  }
  print(numero) #vamos imprimiendo los valores del vector
  numero <- numero-1  #Le vamos restando un 1 en cada ciclo. Recuerden que comenzamos en 20
}
[1] 20
[1] 19
[1] 18
[1] 17
[1] 16
[1] "Se detiene cuando es igual a 15"

Esta es otra forma de hacerlo, pero en lugar de inicializar el objeto numero en 20 e ir restando en cada ciclo, inicializamos en 6 (porque la condicin es que sea mayor a 5), y vamos sumando 1 en cada ciclo.

numero <- 6

while(numero >5){
  if(numero==15){
    break
  }
  print(numero)
  numero <- numero+1
}
[1] 6
[1] 7
[1] 8
[1] 9
[1] 10
[1] 11
[1] 12
[1] 13
[1] 14

Usando next

Se usa next para saltarnos una iteracin en un bucle. Cuando la condicin se cumple, esa iteracin es omitida.

Por ejemplo:

for(i in 1:5){
  if(i==3){
    print("Se ha saltado el nmero 3")
    next   #implementamos el salto
  }
  print(i)  #vamos imprimiendo los valores del vector
}
[1] 1
[1] 2
[1] "Se ha saltado el nmero 3"
[1] 4
[1] 5

Este es lo mismo, pero implementado en un while

x <- 0                 #Creamos un objeto con el valor 0
while(x <5){           #la condicin es que x sea menor a 5
  x <- x+1             #implementamos la suma de 1 en cada ciclo
  if(x==3){
    print("Se ha saltado el nmero 3")  #Mostramos mensaje en cnsola
    next                                #implementamos el salto
  }
  print(x)              #vamos imprimiendo los valores del vector
}
[1] 1
[1] 2
[1] "Se ha saltado el nmero 3"
[1] 4
[1] 5

5.- switch

En lneas generales, esta estructura de control ejecuta una accin o bloque de cdigo segn el valor de un objeto o variable. Es una alternativa al uso de mltiples sentencias if-else anidadas.

La funcin switch evala un ndice o una cadena de texto y elige una opcin entre una lista de alternativas.

La funcin switch toma como entrada un objeto que puede ser un nmero o una cadena de caracteres y una lista de alternativas.

Por ejemplo:

valor <- 1  #definimos un objeto con un nmero 1

#implelemtamos el switch
switch (valor,             #Aqu recoge el valor del objeto
  "Primera accin",        #primera alternativa si el valor es igual a 1
  "Segunda accin",        #segunda alternativa si el valor es igual a 2
  "Tercera accin"         #tercera alternativa si el valor es igual a 3
)
[1] "Primera accin"

Como valor es igual a 1 es resultado es Primera accin. Si cambiamos el valor 2 o 3, el resultado tambin cambiar de acuerdo a las alternativas dispuestas en el switch.

Otro ejemplo un poco ms complejo puede ser la operacin matemtica de dos valores: Si seleccionamos s se realizar la suma, si seleccionamos r, realizaremos una resta y si seleccionamos m, haremos una multiplicacin.

# Definimos dos objetos numricos para poder operar
valor1 <- 4
valor2 <- 2

# Definimos un objeto que sirva de seleccin de alternativas
opcion <- "s"

#implelemtamos el switch
switch (opcion,
  "s" = paste("Suma:", valor1+valor2),
  "r" = paste("Resta:", valor1-valor2),
  "m"= paste("Multiplicacin:",valor1*valor2)
)
[1] "Suma: 6"

Como opcin es igual a s, el resultado es: Suma: 6

EXTRA: Conjunto de Funciones apply

La familia apply pertenece al paquete base R y esta formado por un conjunto de funciones que nos permiten manipular una seleccin de elementos en matrices, arrays, listas y dataframes de forma repetitiva.

Estas funciones nos permiten iterar por los elementos sin tener la necesidad de utilizar explcitamente una funcin iterativa.

Toman como entrada una lista, matriz o vector y aplican esta funcin a cada elemento

La familia apply esta compuesta de las funciones:

  • apply
  • lapply
  • sapply
  • vapply
  • mapply
  • rapply
  • tapply

Funcin apply()

Es una herramienta que permite aplicar una funcin (como sum, mean, min, max, etc), a las filas o columnas de una matriz o data frame, evitando los bucles como el for.

Los argumentos que necesita la funcin son estos:

  • X: La matriz o data frame a la que se aplicar la funcin.

  • MARGIN: Un nmero que indica la direccin de la operacin: 1: Aplica la funcin a cada fila. 2: Aplica la funcin a cada columna.

  • FUN: La funcin que se ejecutar. Se escribe el nombre de la funcin sin parntesis, ya que apply() se encargar de los argumentos.

Por ejemplo, si tenemos el siguiente dataframe:

df = data.frame("sexo"=c("fem", "mas",  "fem",  "mas",  "mas",  "fem"), 
                "edad"=c(30, 18, 20, 21, 20, 30),
                "ingreso"=c(289, 295.5, 290.2, 292,  298, 284.8),
                "gastos"=c(15.4, 14.5, 16.2, 15.9,  16.6, 14.8))

Queremos sumar los valores de las columnas edad, ingreso y gastos:

apply(df[,2:4], 2, sum)   #indicamos por indexacin que solo necesitamos las columnas de la 2 a 4
   edad ingreso  gastos 
  139.0  1749.5    93.4

Ahora queremos el promedio o media de las mismas columnas

apply(df[,2:4], 2, mean)
     edad   ingreso    gastos 
 23.16667 291.58333  15.56667
